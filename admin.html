Admin Dashboard.

Browser-based CMS-lite panel. Edit key content here, preview changes instantly in this browser, and export updated JSON to publish via GitHub.

Edit Rides JSON

Each ride supports: name, category, description, thrill, bestFor, bestTime, emoji.

Edit Store Items JSON

Each store item supports: name, price, description, emoji.

Actions

Save Local Changes updates the site instantly in this browser. Download Updated JSON gives you a file you can upload to GitHub to publish changes live.

How to publish

1. Edit here 2. Click "Download Updated JSON" 3. Replace site-data.json in GitHub 4. Commit changes 5. Cloudflare redeploys

Limit

This is a practical static-site backend. A true multi-user persistent backend would use Supabase, Firebase, or a Git CMS like Decap.
